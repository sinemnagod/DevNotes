DevNotes ✨

DevNotes turns your Markdown into clean HTML instantly. Write on the left, preview on the right, and copy the HTML in one click, with autosave to protect your notes between refreshes.

DevNotes Shortcuts

  • Cmd/Ctrl + Shift + C Copy HTML
  • Cmd/Ctrl + S Export current note as .md
  • Cmd/Ctrl + F Focus find
  • Tab Insert indentation in editor
  • Esc Close this dialog
